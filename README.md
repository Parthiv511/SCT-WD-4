# To-Do Web Application

website link: https://sct-wd-4-o52v.onrender.com
## 📌 Task Objective

Develop a basic to-do web application that enables users to add and maintain tasks.

Allow users to:

- Add tasks
- Organize items in lists
- Mark tasks as completed
- Edit tasks
- Set date & time for tasks

---

# 🚀 Project Overview

This project is a modern and responsive To-Do Web Application built using:

- HTML5
- CSS3
- JavaScript

The application helps users manage daily tasks efficiently with an interactive and user-friendly interface.
